# Awesome Comunità OGD Svizzera

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

[English](README.md) · [Deutsch](README.de.md) · [Français](README.fr.md) · [Italiano](README.it.md) · [Rumantsch](README.rm.md)

Una lista curata di fantastici progetti, strumenti e librerie open-source per lavorare con i dati aperti del governo (OGD) svizzero. Lo scopo di questo progetto è mettere in evidenza contributi di alta qualità da sviluppatori individuali e creatori indipendenti che spesso mancano in altre liste.

## Politica & Democrazia

*   **[politanch/swissdd](https://github.com/politanch/swissdd)** (R) - Dati in tempo reale sulle votazioni federali e cantonali (Democrazia Diretta Svizzera).
*   **[zumbov2/swissparl](https://github.com/zumbov2/swissparl)** (R) - Interfaccia ai servizi web del Parlamento svizzero.
*   **[metaodi/swissparlpy](https://github.com/metaodi/swissparlpy)** (Python) - Wrapper per l'API del Parlamento svizzero per Python.
*   **[zumbov2/SwissParliamentarySpeeches](https://github.com/zumbov2/SwissParliamentarySpeeches)** (Dati) - Raccolta di discorsi nel Parlamento svizzero dal 1999.
*   **[zumbov2/votemapswitzerland](https://github.com/zumbov2/votemapswitzerland)** (R) - Una versione svizzera della famosa visualizzazione «Land doesn't vote, people do.»
*   **[OpenParlData.ch](https://openparldata.ch/)** – L'[API](https://api.openparldata.ch/documentation) offre dati armonizzati su attori politici, atti parlamentari, decreti, consultazioni, votazioni e altro ancora da [78](https://admin.openparldata.ch/#/bodies) parlamenti nazionali, cantonali e comunali.

## Geosapziale & Mappatura

### Librerie & Plugin
*   **[piMoll/swissgeodownloader](https://github.com/piMoll/swissgeodownloader)** (QGIS) - Plugin QGIS per scaricare geodati svizzeri.
*   **[rkaravia/Leaflet.TileLayer.Swiss](https://github.com/rkaravia/Leaflet.TileLayer.Swiss)** (JS) - Plugin per visualizzare le tessere delle mappe svizzere.
*   **[rkaravia/swissgrid](https://github.com/rkaravia/swissgrid)** (JS) - Trasformazione di coordinate tra i sistemi di coordinate proiettate svizzere (LV95/LV03) e WGS 84.
*   **[zumbov2/RSwissMaps](https://github.com/zumbov2/RSwissMaps)** (R) - Tracciare e salvare mappe svizzere tematiche personalizzate con R.
*   **[ValentinMinder/Swisstopo-WGS84-LV03](https://github.com/ValentinMinder/Swisstopo-WGS84-LV03)** (Python) - Script per convertire coordinate tra WGS84 e Svizzero LV03.
*   **[antistatique/swisstopo](https://github.com/antistatique/swisstopo)** (PHP) - Libreria PHP per convertire GPS (WGS84) in coordinate svizzere (LV03/LV95).
*   **[lolochristen/OpenLayers.Blazor](https://github.com/lolochristen/OpenLayers.Blazor)** (C#) - Componente mappa per Blazor con supporto per mappe swisstopo.
*   **[rnckp/opendata_thematic-mapping](https://github.com/rnckp/opendata_thematic-mapping)** (Python) - Guida e codice per mappatura tematica con geodati svizzeri.

### Dati & Confini
*   **[interactivethings/swiss-maps](https://github.com/interactivethings/swiss-maps)** - Geodati svizzeri come TopoJSON.
*   **[mikpan/ch-maps](https://github.com/mikpan/ch-maps)** - Geodati delle mappe svizzere (Generatore GeoJSON/TopoJSON).
*   **[rkaravia/swiss-boundaries-geojson](https://github.com/rkaravia/swiss-boundaries-geojson)** - swissBOUNDARIES3D come GeoJSON.
*   **[cividi/ch-municipalities](https://github.com/cividi/ch-municipalities)** - Lista dei comuni svizzeri come GeoJSON, CSV e JSON.
*   **[SwissStatsR/swissMunicipalities](https://github.com/SwissStatsR/swissMunicipalities)** (R) - Ottenere liste storicizzate ufficiali dei comuni svizzeri dall'UST.

## Statistica & Dati Ufficiali

*   **[lgnbhl/BFS](https://github.com/lgnbhl/BFS)** (R) - Cercare e scaricare dati dall'Ufficio federale di statistica.
*   **[lgnbhl/I14Y](https://github.com/lgnbhl/I14Y)** (R) - Cercare e ottenere metadati dalla piattaforma di interoperabilità I14Y della Svizzera.

## Utilità & Servizi

*   **[OpendataCH/Transport](https://github.com/OpendataCH/Transport)** (PHP) - API dei trasporti pubblici svizzeri.
*   **[metaodi/openerz](https://github.com/metaodi/openerz)** (API) - OpenERZ è un'API aperta per la raccolta dei rifiuti di molti comuni svizzeri diversi.
*   **[validitylabs/zefix](https://github.com/validitylabs/zefix)** (TypeScript) - Wrapper sottile attorno all'API REST ZEFIX per cercare e recuperare dati aziendali.
*   **[abertschi/postcard_creator_wrapper](https://github.com/abertschi/postcard_creator_wrapper)** (Python) - Un wrapper python attorno all'API REST del creatore di cartoline svizzere.
*   **[OneGov/onegov-cloud](https://github.com/OneGov/onegov-cloud)** (Python) - Potente framework open-source per applicazioni web per comuni svizzeri e ONG.
*   **[rnckp/starter-code-ktzh](https://github.com/rnckp/starter-code-ktzh)** (Python/R) - Codice di avvio per analizzare set di dati CSV dall'Ufficio di statistica del Canton Zurigo.
*   **[TrainDelayVisualizer/TrainDelayVisualizer](https://github.com/TrainDelayVisualizer/TrainDelayVisualizer)** (TypeScript) - Applicazione per visualizzare ritardi dei treni da opentransportdata.swiss.
*   **[cevi/automatic_walk-time_tables](https://github.com/cevi/automatic_walk-time_tables)** (Python) - Genera tabelle dei tempi di percorrenza da tracce Swisstopo o file GPX.
