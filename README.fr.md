# Awesome Communauté OGD Suisse

[![Awesome](https://awesome.re/badge.svg)](https://awesome.re)

[English](README.md) · [Deutsch](README.de.md) · [Français](README.fr.md) · [Italiano](README.it.md) · [Rumantsch](README.rm.md)

Une liste organisée de projets, outils et bibliothèques open-source géniaux pour travailler avec les données publiques ouvertes (OGD) suisses. Le but de ce projet est de mettre en valeur les contributions de haute qualité des développeurs individuels et des créateurs indépendants qui manquent souvent sur d'autres listes.

## Politique & Démocratie

*   **[politanch/swissdd](https://github.com/politanch/swissdd)** (R) - Données en temps réel sur les votations fédérales et cantonales (Démocratie directe suisse).
*   **[zumbov2/swissparl](https://github.com/zumbov2/swissparl)** (R) - Interface vers les services web du Parlement suisse.
*   **[metaodi/swissparlpy](https://github.com/metaodi/swissparlpy)** (Python) - Wrapper pour l'API du Parlement suisse pour Python.
*   **[zumbov2/SwissParliamentarySpeeches](https://github.com/zumbov2/SwissParliamentarySpeeches)** (Données) - Collection de discours au Parlement suisse depuis 1999.
*   **[zumbov2/votemapswitzerland](https://github.com/zumbov2/votemapswitzerland)** (R) - Une version suisse de la célèbre visualisation «Land doesn't vote, people do.»
*   **[OpenParlData.ch](https://openparldata.ch/)** – L'[API](https://api.openparldata.ch/documentation) offre des données harmonisées sur les acteurs politiques, les objets parlementaires, les décrets, les consultations, les votes et plus encore provenant de [78](https://admin.openparldata.ch/#/bodies) parlements nationaux, cantonaux et communaux.

## Géospatial & Cartographie

### Bibliothèques & Plugins
*   **[piMoll/swissgeodownloader](https://github.com/piMoll/swissgeodownloader)** (QGIS) - Plugin QGIS pour télécharger des géodonnées suisses.
*   **[rkaravia/Leaflet.TileLayer.Swiss](https://github.com/rkaravia/Leaflet.TileLayer.Swiss)** (JS) - Plugin pour afficher les tuiles de cartes suisses.
*   **[rkaravia/swissgrid](https://github.com/rkaravia/swissgrid)** (JS) - Transformation de coordonnées entre les systèmes de coordonnées suisses (LV95/LV03) et WGS 84.
*   **[zumbov2/RSwissMaps](https://github.com/zumbov2/RSwissMaps)** (R) - Tracer et sauvegarder des cartes suisses thématiques personnalisées avec R.
*   **[ValentinMinder/Swisstopo-WGS84-LV03](https://github.com/ValentinMinder/Swisstopo-WGS84-LV03)** (Python) - Scripts pour convertir les coordonnées entre WGS84 et le système suisse LV03.
*   **[antistatique/swisstopo](https://github.com/antistatique/swisstopo)** (PHP) - Bibliothèque PHP pour convertir les coordonnées GPS (WGS84) en coordonnées suisses (LV03/LV95).
*   **[lolochristen/OpenLayers.Blazor](https://github.com/lolochristen/OpenLayers.Blazor)** (C#) - Composant cartographique pour Blazor avec support des cartes swisstopo.
*   **[rnckp/opendata_thematic-mapping](https://github.com/rnckp/opendata_thematic-mapping)** (Python) - Guide et code pour la cartographie thématique avec des géodonnées suisses.

### Données & Frontières
*   **[interactivethings/swiss-maps](https://github.com/interactivethings/swiss-maps)** - Géodonnées suisses au format TopoJSON.
*   **[mikpan/ch-maps](https://github.com/mikpan/ch-maps)** - Géodonnées des cartes suisses (Générateur GeoJSON/TopoJSON).
*   **[rkaravia/swiss-boundaries-geojson](https://github.com/rkaravia/swiss-boundaries-geojson)** - swissBOUNDARIES3D au format GeoJSON.
*   **[cividi/ch-municipalities](https://github.com/cividi/ch-municipalities)** - Liste des communes suisses aux formats GeoJSON, CSV et JSON.
*   **[SwissStatsR/swissMunicipalities](https://github.com/SwissStatsR/swissMunicipalities)** (R) - Obtenir des listes officielles historicisées des communes suisses de l'OFS.

## Statistiques & Données Officielles

*   **[lgnbhl/BFS](https://github.com/lgnbhl/BFS)** (R) - Rechercher et télécharger des données de l'Office fédéral de la statistique.
*   **[lgnbhl/I14Y](https://github.com/lgnbhl/I14Y)** (R) - Rechercher et obtenir des métadonnées de la plateforme d'interopérabilité I14Y de la Suisse.

## Utilitaires & Services

*   **[OpendataCH/Transport](https://github.com/OpendataCH/Transport)** (PHP) - API des transports publics suisses.
*   **[metaodi/openerz](https://github.com/metaodi/openerz)** (API) - OpenERZ est une API ouverte pour la collecte des déchets de nombreuses communes suisses.
*   **[validitylabs/zefix](https://github.com/validitylabs/zefix)** (TypeScript) - Wrapper léger autour de l'API REST ZEFIX pour rechercher et récupérer des données d'entreprise.
*   **[abertschi/postcard_creator_wrapper](https://github.com/abertschi/postcard_creator_wrapper)** (Python) - Un wrapper python autour de l'API REST du créateur de cartes postales suisses.
*   **[OneGov/onegov-cloud](https://github.com/OneGov/onegov-cloud)** (Python) - Framework d'application web open-source puissant pour les communes suisses et les ONG.
*   **[rnckp/starter-code-ktzh](https://github.com/rnckp/starter-code-ktzh)** (Python/R) - Code de démarrage pour l'analyse des jeux de données CSV de l'Office statistique du canton de Zurich.
*   **[TrainDelayVisualizer/TrainDelayVisualizer](https://github.com/TrainDelayVisualizer/TrainDelayVisualizer)** (TypeScript) - Application pour visualiser les retards de trains à partir de opentransportdata.swiss.
*   **[cevi/automatic_walk-time_tables](https://github.com/cevi/automatic_walk-time_tables)** (Python) - Génère des tableaux de temps de marche à partir de traces Swisstopo ou de fichiers GPX.
